Commit message format: `type(scope): subject`

Examples:
```
chore(nav): wire ScamResultCardScreen to real AnalysisResponse via route args
feat(chat): add voice input via Whisper transcription
fix(ui): consolidate duplicate RiskLevel enum to analysis_response.dart
```

- `type`: `feat` (new user-facing feature), `fix` (bug), `chore` (wiring/refactor),
  `ci` (CI config), `refactor` (internal restructure), `docs` (docs only)
- `scope`: the area — `nav`, `chat`, `api`, `ui`, `auth`, `providers`, `models`
- Subject: imperative, lowercase, ≤ 72 chars, no trailing period
